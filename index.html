数字的姓氏

发表于 2016-03-15

日本人的姓氏成千上万，其中不乏有趣的，奇怪的姓氏。
在刚学日语的时候，有一次遇到一个名叫“御手洗（みたらい）”的日本人，我开口就叫“おてあらい（卫生间的意思）”さん。事后回想，当时没被揍，可能是得益于日本人内敛的性格吧。

STP/RSTP学习笔记

发表于 2016-03-01

此处收录一些与STP/RSTP相关的名词及其解释。

No. Terminology Explanation
1 Inferior BPDU An inferior BPDU is the one carries information about the root bridge that is worse than the one currently stored for the port.
2 Superior BPDU A superior BPDU is the one carries information about the root bridge that is better than the one currently stored for the port.
3 Root Port It receives the best BPDU on a bridge.
4 Designated Port It sends the best BPDU on the segment to which it is connected.
5 Alternate Port It receives more useful BPDUs from another bridge and is a port block.
6 Backup Port It receives more useful BPDUs from the same bridge it is on and is a port blocked.
